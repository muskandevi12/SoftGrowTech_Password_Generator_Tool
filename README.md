# SoftGrowTech Password Generator Tool

This is a Python GUI-based Password Generator Tool.

## Features

* Generates secure random passwords
* Includes letters, numbers, and special characters
* Allows users to choose password length
* Uses a while loop until the user is satisfied with the password
* Built using Python Tkinter

## Technologies Used

* Python
* Tkinter
* Random module
* String module

## How to Run

1. Open the project folder in VS Code.
2. Run the Python file:

```bash
python password_generator_gui.py
```

## Project Requirement

Create a Python program that generates secure random passwords.
